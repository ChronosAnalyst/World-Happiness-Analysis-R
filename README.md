# World Happiness Report Analysis (2005-2022)

This repository contains my Capstone Project for the **Data Analyst** certification at the **National and Kapodistrian University of Athens**.

## Project Overview
An in-depth statistical analysis of global happiness trends using R. The project explores socioeconomic drivers of well-being through:
- **Exploratory Data Analysis (EDA)**: Visualizing correlations and trends.
- **Multiple Linear Regression**: Quantifying the impact of GDP, social support, and healthy life expectancy.
- **Unsupervised Machine Learning**: Using **k-means clustering** to categorize countries into 3 distinct well-being tiers.

## Tech Stack
- **Language:** R
- **Key Libraries:** `tidyverse`, `ggplot2`, `cluster`, `factoextra`

## Key Findings
- Economic prosperity and social support are the strongest predictors of subjective well-being.
- Identified 3 distinct global clusters, revealing significant disparities in institutional trust and quality of life.

## How to run
1. Open `capstone_happiness.Rproj` in RStudio.
2. Run scripts in order from the `scripts/` folder.
